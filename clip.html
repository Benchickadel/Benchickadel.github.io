Clip_V5

This was iteration #5.  I originally based this on a rivet clip (see the other picture on right).  I was able to make one or two sets work but I couldn't get the tolerances correct and possibly PLA doesn't have the flex necessary.  This version is based on an automotive metal clip. I modeled the sliding ring separately and used grasshopper to create a box array to make printing easier.  The clip is adjustable along the z access using the multiple stacked torus solids to create the ridges.  A solid cylinder is running up the center.  The base is a separate brep that gets union-ed when baked. Another mode would be to make a point to poke through the cardboard but it makes them dangerous so a pointy poking tool would be good. I used a pen to make my structure.

Clip_V1-4

This was iteration1-4.  I added ridges to this model to create a ratcheting action but they just kept breaking and were hard to use.  Ended up scraping this idea after I printed 20 pairs of them and broke probably all of them.

3D printing Clip

Settings/material

  • Standard Default Cura 4.6
  • infill 20%
  • PLA
  • ~7hr print

 

The one thing I took away while printing is that creating large multiples and organizing the bed of prints in Cura isn't easy.  Creating the array in Grasshopper saved time.  I was able to print 36 posts with a little room to spare on the Ender 3 Pro. I/ve been lucky enough to have two printers at home and printed the orange clip on a MethodX.  I went into production mode and printed about 64 of these things.  I realized that the ring would benefit being slightly smaller in diameter to make it a bit stronger or exaggerate the "grooves" on the post piece.  For now it works!
