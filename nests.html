7 sided taper

Big shout out to Jasper for his help on this one. I was stuck on this one. The videos helped tremendously! I did want to attempt the telescoping effect.  Found the taper block and watched a quick input tutorial and it worked great.

Nested printing

Settings/material

  • Standard Default Cura 4.6
  • infill 20%
  • PLA
  • ~5hr print time

The quality of the outer walls could have been better.  My first attempt on a different model got stuck together on the z-seam.  I did set a random start to each layer and that seemed to help tons. when I pulled it off the bed it "telescoped" out. I didn't add a bottom because I really did not want to (time was a factor as well) remove supports
