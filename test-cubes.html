Ultimake Cura 4.5

I am using Ultimake Cura 4.5 for all of my slicing. You can download it here

For my 20mm cube, I created a 20mmx20mmx20mm cube in rhino6 and added an embossed 'key' to identify the front and top of each print.  This reduced how much I needed to draw on each cube.  I did mark the bottom of each print as well to number them for my own tracking and note taking.

Low Quality

Settings and Results.

Material: PLA

Build Plate Temp: 200˚C

Nozzle Temp: 50˚C

layer height: 0.28mm

Infill: 35%

Top/bottom layers patter: concentric

Print time: 27min

Actual HxLxW (mm): 19.8 x 20.2 x 20.1

Comments: I was surprised the actual dimensions were so close to original measurements. I accidentally left the infill at 35% instead of using the default 20%.

Standard Quality

Settings and Results.

Material: PLA

Build Plate Temp: 200˚C

Nozzle Temp: 50˚C

layer height: 0.2mm

Infill: 10%

Top/bottom layers patter: concentric

Print time: 26min

Actual HxWxL: 19.94 x 20.08 x 20.19

Comments: I changed the infill on this one and I think that is why there are spaces between extrusions on the top layer.  Also the concentric pattern does not help fix these spaces and can almost exaggerate them by stacking the pattern and not closing gaps. I think if I had used a linear top pattern it would have alternated the patterns helping eliminate the gaps. This would also strengthen the print (as in plywood or bricklaying).

Dynamic Quality

Settings and Results.

Super Quality

Settings and Results.

Material: PLA

Build Plate Temp: 200˚C

Nozzle Temp: 50˚C

layer height: 0.1

Infill: 10%

Top/bottom layers patter: lines

Print time: 44min

Actual HxWxL: 19.94 x 20.01 x 19.97

Comments: pretty darn accurate at +/- 0.06mm.  Depending on what's needed, I'd probably print in standard quality to save time/material while printing.

20mm test cube with 5mm hole

(left to right)

Cube1, Cube 2, Cube 2

Settings and Results.

Material: PLA

Build Plate Temp: 200˚C

Nozzle Temp: 50˚C

Default Dynamic setting

layer height: 0.2

Infill: 10%

Top/bottom layers patter: zigzag

Print time: ~2hrs

Actual HxWxL x ID ( mm):

Cube1: 19.90 x 20.06 x 20.02 x 4.89id

Cube2: 19.99 x 20.06 x 19.97 x 5.09id

Cube3: 19.91 x 20.10 x 20.03 x 5.59id

 

Comments: In Rhino6, I created 3 cubes and boolean subtracted a cylinder from each cube.  I slightly increased the diameter of each cylinder to print all the cubes simultaneously, hoping to get the correct diameter from one of the cubes (just how photographers used to bracket settings with film!).  Cube1 had  a diameter hole of 5mm. Cube 2 had a diameter of 5.2mm and Cube3 had a diameter of 5.4mm. Above is the actual measurements. Cube 2 is the closest to the +/- 0.5mm threshold for all dimensions but Cube 1 is also a possible choice
